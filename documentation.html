Hannibal Website Documentation

1. Overview

This website is a gallery and information site for the TV show Hannibal. It features a dark theme with a focus on displaying images from the show, along with character information, quotes, and background music.

2. File Structure

    mind-palace/
    ├── index.html
    ├── style.css
    ├── script.js
    ├── documentation.html
    └── assets/
        ├── background.jpg
        ├── background-music.mp3
        └── image_0.png to image_17.png

3. HTML Structure (index.html)

The main HTML file contains the following sections:

  • Header with navigation menu
  • Main content:
    • Gallery section
    • About section
    • Characters section
    • Quotes section
    • Legacy section
  • Footer with creator information and documentation link
  • Audio element for background music

4. CSS Styles (style.css)

The CSS file defines the visual style of the website, including:

  • Dark theme with blood red accents
  • Responsive design for mobile devices
  • Image carousel and lightbox styles
  • Blurred background effect

5. JavaScript Functionality (script.js)

The JavaScript file handles the interactive elements of the site:

  • Mobile menu toggle
  • Image carousel functionality
  • Lightbox for enlarged image viewing
  • Dynamic character list population
  • Random quote generator
  • Smooth scrolling for navigation
  • Background music control

6. How to Add New Content

Adding Images:

  1. Add new image files to the assets/ directory
  2. Update the images array in script.js with new file names

Adding Quotes:

  1. Add new quote strings to the quotes array in script.js

Adding Characters:

  1. Add new character objects to the characters array in script.js

7. Customization

To customize the website:

  • Modify color variables in the :root selector in style.css
  • Update background image by replacing assets/background.jpg
  • Change background music by replacing assets/background-music.mp3

8. Browser Compatibility

This website is designed to work with modern browsers. For older browsers, some features may not function correctly.

9. Known Issues and Limitations

  • Autoplay of background music may be blocked by some browsers
  • Image lazy loading is not implemented, which may affect initial load times

10. Future Improvements

  • Implement lazy loading for gallery images
  • Add more interactive elements like character bios or episode guides
  • Enhance the lightbox with touch gestures for mobile users
  • Implement a search functionality for quotes or characters
